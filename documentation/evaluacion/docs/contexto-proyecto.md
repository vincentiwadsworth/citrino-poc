# Contexto del Proyecto Citrino

## Antecedentes

Citrino es una empresa de intermediación inmobiliaria que opera con datos dispersos en múltiples hojas de cálculo. Julio (CEO de Citrino) expresó específicamente la necesidad de poder "chatear con los datos" como interfaz principal de consulta.

## Problemática Actual

* Cientas de hojas de cálculo sin estandarización
* Datos duplicados e inconsistentes
* Imposibilidad de análisis transversal
* Dependencia de consultas manuales

## Objetivo a Mediano Plazo

Crear un sistema unificado que permita:

1. Consultas conversacionales en lenguaje natural
2. Análisis de mercado en tiempo real
3. Recomendaciones inteligentes de propiedades
4. Reporting automático personalizado

## Stakeholders Clave

* CEO: Interfaz conversacional como prioridad
* Agentes inmobiliarios: Acceso rápido a información
* Analistas: Capacidad de reporting avanzado
